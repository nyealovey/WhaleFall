# Prompts 与协作模板

> 状态：Draft

本目录用于存放可复用的提示词与协作模板（评审清单、专项分析提示等）。

## 约定

- 新增文件名统一使用英文 `kebab-case.md`。
- 单次会话的临时产物不要长期堆放在这里：
  - 若与具体变更绑定，放到 `changes/`。
  - 若是评审结论，放到 `reports/`。
  - 过期内容进入 `_archive/`。
