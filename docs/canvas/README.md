# Canvas

- 生成规范(真源): `../standards/json-canvas-generation-standard.md`

- 模板:
  - 仓库内参考模板: `docs/canvas/template.canvas`
  - Obsidian vault 模板: `docs/Obsidian/canvas/template.canvas`
