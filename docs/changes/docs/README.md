# Docs 变更

> 状态: Active

本目录用于记录 \"文档新增/调整/迁移/规范更新\" 等变更, 与 PR/Issue 对齐.

## 命名规则

- 新增文档文件名必须使用三位编号前缀(从 `001` 开始递增): `NNN-short-title.md`.
- 多 PR / 多阶段推进的事项建议拆分为:
  - `NNN-short-title-plan.md`
  - `NNN-short-title-progress.md`
